API для аутентификации пользователей
Этот проект предоставляет RESTful API для регистрации и управления пользователями с поддержкой двух методов аутентификации: по email и через Telegram.

Возможности
✅ Двойной метод аутентификации (Email + Telegram)

🔐 Безопасное хеширование паролей с использованием bcrypt

🛡️ Валидация входных данных с помощью Pydantic моделей

🗃️ Использование PostgreSQL в качестве базы данных

🚀 Асинхронный ввод-вывод с SQLAlchemy 2.0 и FastAPI
